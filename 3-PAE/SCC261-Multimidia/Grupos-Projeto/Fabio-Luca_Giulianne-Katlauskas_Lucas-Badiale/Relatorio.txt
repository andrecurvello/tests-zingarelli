Descrição do trabalho:

O projeto está separado em 2 partes: Compressão e descompressão.

De acordo com os dados contidos no cabeçalho do arquivo BMP é estabelecido o número de blocos necessários para armazenar as informações do arquivo BMP. Os blocos são estruturas com 3 matrizes (R, G e B) de 8 por 8 e um ponteiro, que aponta para o próximo bloco da imagem. Ou seja, é formada por uma lista de blocos. 

É realizado o level shift dos dados contidos em cada bloco, de forma a deixar os valores entre um novo intervalo. 

Para posterior leitura do arquivo é salvo em uma tabela o número de bits significativos a serem lidos para um determinado valor.

Sabendo o número de bits significativos para cada valor, estes são convertidos para bits e salvos no arquivo binário.
